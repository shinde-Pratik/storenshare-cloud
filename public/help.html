  • Upload File

  • How To Upload A File?

    Once you enter on home page you can see the 2 options there.
    1. On Navigation Bar select Upload File menu to redirected on Upload page.
    2. On Home page By pressing Upload File button you will redirected to Upload page.

    Once you redirected to upload page you will able to upload your file.
    By Drag & Drop file to upload Container
    By clicking 'browse' button available on upload container.

  • Where My Files Go?

    File you uploaded are stored in cloud based database and the link to it will provided to you. These files are only accessible for 24hrs and link will expired.

  • Can I Upload Multiple Files?

    You can Upload Multiple files only of size upto 100Mb at single time

  • Share File

  • How To Share Files?

    Once you upload file to Upload Container the link will provided to you.You can share the link with receiver by various communication chanels.And by specifying sendors and receivers mail you can send it through mail also.

  • How Much User Can Access My Link?

    If you share file using link then multiple users can access it.And if you share with mail then the single receiver will get the mail.

  • How Long We Can Access Link?

    Receiver can acess link only for 24Hrs for he receives the link.
